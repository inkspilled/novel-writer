# Novel Writer - AI 小说写作桌面应用

多 Agent 协作的小说创作工具，从立意到定稿全流程覆盖。

## 启动

```bash
cd novel-writer
pip install -e .
python -m novel_writer
```

## 写作流程

| 阶段 | Agent | 职责 |
|------|-------|------|
| 立意 | 📋 主编 | 题材、风格、目标读者、核心主题 |
| 大纲 | 🗺️ 策划 | 章节结构、人物设定、世界观、伏笔 |
| 初稿 | ✍️ 写手 | 按大纲逐章写作 |
| 校对 | 🔍 校对 | 错别字、语法、标点、一致性 |
| 审核 | ✅ 审核 | 剧情逻辑、人物一致性、节奏 |
| 灵感 | 💡 灵感 | 创意激发、脑洞拓展、题材联想 |
| 润色 | ✨ 润色 | 文笔提升、场景描写、情感表达 |

支持自定义 Agent，可配置专属模型、技能范围和系统提示词。

## 项目目录结构

每个小说项目独立存储为一个目录：

```
data/projects/{project_name}/
├── meta.json                    # 项目元信息（标题、题材、风格、目标字数）
├── chat.db                      # 聊天记录（项目级 SQLite）
├── planning/                    # 规划文档（.md）
│   ├── 立意.md
│   ├── 大纲.md
│   ├── 人物设定.md
│   ├── 世界观.md
│   ├── 时间线.md
│   ├── 主线.md
│   ├── 支线.md
│   └── 伏笔.md
├── chapters/                    # 章节正文（.txt）
│   ├── 1_章节名.txt
│   ├── 1_章节名.outline.md      # 细纲（可选，.md）
│   └── ...
├── inspiration/                 # 灵感记录（.md）
│   └── 3_灵感.md
├── review/                      # 审校产出（.md）
│   ├── 审核报告.md
│   └── 校对报告.md
└── workflow.json                # 工作流进度（断点恢复用）
```

章节正文用 `.txt`（纯文本），规划文档和灵感用 `.md`。章节序号不补零，支持任意位数（`1_xxx.txt` 到 `12389127_xxx.txt`）。

## 界面

```
┌──────────┬──────────────────┬───────────────────────────┐
│  侧边栏   │     编辑区        │   智能体办公室（占 50%+）   │
│          │                  │                           │
│ 项目管理   │  正文 / 大纲 / 备注 │  📋🗺️✍️🔍✅💡✨（30%）    │
│ 章节树    │  实时字数统计      │  办公室场景 + 动画          │
│ 字数统计   │                  │  工作流进度条              │
│          │                  │  💬 对话区（70%）           │
└──────────┴──────────────────┴───────────────────────────┘
```

## 智能体办公室

每个 Agent 在虚拟办公室里有自己的工位和个性行为：

| 状态 | 行为 |
|------|------|
| 空闲 | 喝咖啡、打盹、刷手机、发呆看窗外 |
| 工作中 | 敲键盘，屏幕有代码滚动 |
| 等待中 | 端咖啡杯、走廊踱步 |
| 完成 | 伸懒腰、竖大拇指 |
| 出错 | 抱头、拍桌子 |
| 摸鱼 | 打游戏、睡觉、翻墙溜号 |

点击办公室里的 Agent 可以查看/继续与该 Agent 的对话。

## Agent 对话

- **项目级隔离**：每个项目独立聊天记录（`chat.db` 存在项目目录下）
- Markdown 渲染：代码块、表格、标题、列表
- 流式输出 + 思考过程展示，切换智能体不影响正在进行的流式输出
- 对话上下文包含项目全部规划文档和所有章节正文
- Ctrl+Enter 发送消息

## 工作流

支持 4 种工作流模式：

| 模式 | 说明 | 步骤 |
|------|------|------|
| 📖 新书 | 从立意到审校全流程 | 12 步（可配置章节数） |
| ✍️ 续写 | 从已有章节继续 | 4 步（自动检测起始章节） |
| 🔍 查漏补缺 | 检查并补写缺失章节 | 按需 |
| ✅ 校验 | 审核 + 校对已有章节 | 2 步 |

- **办公室联动**：工作流执行时，办公室里的 Agent 实时反映状态
- **断点恢复**：中断后自动从上次进度继续
- **章节复用**：已有章节文件会被覆盖写入，不会重复创建

## 菜单

| 菜单 | 功能 |
|------|------|
| 文件 | 新建 / 打开 / 保存项目 |
| 模型与智能体 | 模型设置、智能体管理 |
| 工作流 | 开始工作流（弹出模式选择）、加载默认工作流 |
| 关于 | 外观设置、关于 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建项目 |
| Ctrl+O | 打开项目 |
| Ctrl+S | 保存项目 |
| Ctrl+, | 外观设置 |
| Ctrl+Shift+W | 开始工作流 |
| Ctrl+Enter | 发送消息 |

## 数据存储

| 文件 | 说明 |
|------|------|
| `data/config.json` | 全局配置（主题、语言、模型） |
| `data/projects/*/chat.db` | 项目级聊天记录（SQLite） |
| `data/projects/*/` | 小说项目目录 |
| `config/agents.json` | 智能体配置 |
| `config/default_agents.json` | 智能体默认配置（重置用） |
