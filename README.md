# Novel Writer - AI 小说写作桌面应用

多 Agent 协作的小说创作工具，从立意到定稿全流程覆盖。

## 启动

```bash
cd novel-writer
pip install -e .
python -m novel_writer
```

## 写作流程

| 阶段 | Agent | 职责 |
|------|-------|------|
| 立意 | 📋 主编 | 题材、风格、目标读者、核心主题 |
| 大纲 | 🗺️ 策划 | 章节结构、人物设定、世界观、伏笔 |
| 初稿 | ✍️ 写手 | 按大纲逐章写作 |
| 校对 | 🔍 校对 | 错别字、语法、标点、一致性 |
| 审核 | ✅ 审核 | 剧情逻辑、人物一致性、节奏 |
| 灵感 | 💡 灵感 | 创意激发、脑洞拓展、题材联想 |
| 润色 | ✨ 润色 | 文笔提升、场景描写、情感表达 |

支持自定义 Agent，可配置专属模型、技能范围和系统提示词。

## 项目目录结构

每个小说项目独立存储为一个目录：

```
data/projects/{project_name}/
├── meta.json                    # 项目元信息（标题、题材、风格、目标字数）
├── planning/                    # 规划文档
│   ├── 立意.md                  # 核心立意、卖点、目标读者
│   ├── 大纲.md                  # 总纲/卷纲/章纲
│   ├── 人物设定.md              # 主要人物档案
│   ├── 世界观.md                # 世界观/设定体系
│   ├── 时间线.md                # 故事时间轴
│   ├── 主线.md                  # 主线剧情脉络
│   ├── 支线.md                  # 支线剧情
│   └── 伏笔.md                  # 伏笔埋设/回收记录
├── chapters/                    # 章节正文
│   ├── 001_章节名.md            # 第1章正文
│   ├── 001_章节名.outline.md    # 第1章细纲（可选）
│   └── ...
├── inspiration/                 # 灵感记录
│   └── 001_主题.md
├── review/                      # 审校产出
│   ├── 审核报告.md
│   └── 校对报告.md
└── workflow.json                # 工作流进度（断点恢复用）
```

## Agent 对话

- 每个 Agent 独立聊天记录，SQLite 持久化，切换不丢失，重启自动恢复
- Markdown 渲染：代码块、表格、标题、列表
- 快捷操作栏：根据 Agent 技能自动生成常用指令
- 流式输出 + 思考过程展示，切换智能体不影响正在进行的流式输出
- 选择 Agent 时名称行显示当前使用的模型（专属模型蓝色 / 默认模型灰色）
- Ctrl+Enter 发送消息
- 关闭应用时自动断开所有模型连接，不会残留后台进程

## 模型配置

支持所有 OpenAI 兼容接口的模型供应商，只需 API Key + Base URL + 模型名称。

预置供应商：DeepSeek、Moonshot (Kimi)、智谱 GLM、通义千问、小米 MiMo、OpenAI、Claude (需代理)、Ollama (本地)。

在 `模型设置` 中可以：

- **默认模型**：顶部下拉框选择已保存模型，一键设为默认
- **保存方案**：将当前配置保存为命名模型方案
- **已保存模型**：左侧列表管理，点击编辑，支持删除
- 为每个智能体独立选择不同模型（智能体管理 → 专属模型下拉选择）
- 测试连接是否正常

添加新供应商：编辑 `config/default_providers.json`，或直接在模型设置中手动填写 Base URL。

## 数据存储

| 文件 | 说明 |
|------|------|
| `data/config.json` | 全局配置（主题、语言、模型、智能体） |
| `data/chat.db` | 聊天记录（SQLite） |
| `data/projects/*/` | 小说项目目录 |
| `config/agents.json` | 智能体配置 |
| `config/default_agents.json` | 智能体默认配置（重置用） |
